== Description ==

This Plugin is used to Redirect page to Home on Logout.

* Ony for Subscriber
* Redirect URL for Logout action.
* Custom Redirection for Logout
* Redirect Rule for Subscriber
* Logout Redirect to Homeurl

== Installation ==

1. Upload folder `wpi-custom-logout` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
